Проект состоит из 2-х частей:

1.  Парсинг всех скидочных товаров в магазинах Мвидео (Москва), а так же запись этих данных в БД; *Mvideo_parsing
2. Телеграмм-бот, который на основании данных в БД, при нажатии определенной кнопки, высылает карточки скидочных товаров. При этом присутсвует поиск товаров по вводу с клавиатуры. *Mvideo_bot

Для функционирования проекта необходимо:

1. Развернуть БД на PostgreSQL;
2. Запустить скрипты(create_mv_products.sql, create_usr_clcks.sql) создания таблиц, в которые будет переноситься вся необходимая информация;
3. В скриптах pg_config.py заменить реквизиты подключения к БД на свои, или записать их в переменные среды windows;
4. В скрипте params.py добавить свои cookies сайта, которые можно взять из cURL'a основных GET-запросов в любой категории товаров, к примеру: https://www.mvideo.ru/smartfony-i-svyaz-10/smartfony-205?reff=menu_main;
5. Получить токен бота в "Телеграм", который можно запросить у @BotFather. После чего в скрипте Mvideo_bot\main.py внести его в параметр 'token', либо создать переменную в среде windows, с названием 'TG_MVIDEO_BOT', записав в нее токен.
